# react-redux-router-vote
An attempt for react，Include webpack, Redux, router
#运行
下载安装依赖包 npm install
运行 npm run dev
#打包
运行 npm run build
打包路径可重新定义
``
output:{
        path:path.resolve(__dirname,'../lyr-project/dist'),
        filename:"[name].js",
        chunkFilename:'[chunkhash:8].chunk.js'
    },
``
